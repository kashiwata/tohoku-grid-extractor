# 系統図PDF解析の手法 / Methodology

## 1. 入力データ

東北電力ネットワーク株式会社が公開する **実績系統図PDF**（地域別, 1ページ, 842×1191pt）。

PDFはベクタ描画形式で、以下の描画要素を含む:
- 送電線: 細い黒系の折れ線パス（線幅 1.4–1.9pt）
- 変電所シンボル: 円形（白塗り）
- 発電機シンボル: G文字付き円形
- 文字ラベル: 変電所名・線番号
- 県境線: Bezier曲線（グレー）
- フレーム: 外枠矩形

---

## 2. 送電線パスの抽出

PyMuPDF (`page.get_drawings()`) でPDFの全描画要素を取得し、以下のフィルタを適用:

```
線幅フィルタ:   WIDTH_MIN = 1.4pt ≤ w ≤ 1.9pt = WIDTH_MAX
色フィルタ:     max(R, G, B) < 0.15  (黒系色)
長さフィルタ:   セグメント長 ≥ 3.0pt
Bezierフィルタ: n_pts ≤ 60  (n_pts > 60 は県境曲線として除外)
```

**Bezier フィルタの根拠:**

PDFのBezier曲線は BEZIER_DIV=20 で細分化される（1曲線 = 21点）。
- 送電線の折れ線: 直線セグメントのみ → n_pts ≤ 50
- 県境曲線: 多数のBezier曲線で構成 → n_pts ≥ 421

閾値 60pt により、両者を確実に分離できる。

**大型フレーム除外:**

ページ幅の55%以上・高さの55%以上を占める矩形パスは外枠と判断して除外。

---

## 3. 変電所・発電機シンボルの検出

**変電所（円形シンボル）:**
- `fill` が白色（輝度 > 0.95）
- 近似的に円形（縦横比 0.7–1.3, 面積 ≥ 5pt²）

**発電機シンボル:**
- 円形条件 + 近傍テキストに "G" を含む、または内部に描画あり

**変電所種別の判定:**

テキストラベルの検索半径 R_LABEL=50pt で近傍テキストを取得し、キーワードマッチング:
- 「1次」「一次」 → 1次変電所
- 「開閉所」 → 開閉所
- その他（円シンボル） → 変電所

---

## 4. 接続点クラスタリング（DBSCAN）

抽出した線分セグメントの **端点** を DBSCAN でクラスタリング:

```
eps = 10.0 pt   (クラスタリング半径)
min_samples = 1 (単独点もクラスタ扱い)
```

各クラスタの重心をノード（接続点 `junction`）として登録。

**eps の最適化:**
- eps=3: 107 CC（過小、線分が繋がらない）
- eps=10: 35 CC（最適）
- eps=15: 13 CC（過大、自己ループ多発）
- eps=20:  6 CC（過大）

---

## 5. シンボル→送電線スナップ

変電所・発電機シンボルの中心点を最近傍送電線にスナップする（Step B2）:

```
SNAP_DIST_LINE = 25pt:  線分を分割してノードを挿入
SNAP_FALLBACK  = 70pt:  孤立ノードとして追加
```

スナップ対象から除外された孤立シンボルは、最近傍接続点とスタブ接続。

---

## 6. 地理座標変換

既知の変電所位置（アンカー点）を使った最小二乗アフィン変換により、PDF座標 → 緯度経度を推定:

```
[lon]   [a  b] [x]   [tx]
[lat] = [c  d] [y] + [ty]
```

**精度（RMSE）:**
- 山形県: ≈ 5–10 km
- その他県: ≈ 8–20 km

系統図は模式的な表現のため、地理的精度は ±30% 程度。

---

## 7. 電力モデル構築

### 線路パラメータ（Pi モデル）

```
l_km = l_pdf [pt] × PT_TO_KM [km/pt]

R [Ω] = r_per_km × l_km
X [Ω] = x_per_km × l_km
B [S] = b_per_km × l_km

r_pu = R / Z_base,  X_base = V_base² / S_base [Ω]
x_pu = X / Z_base
b_pu = B × Z_base
```

**PT_TO_KM（地域別スケール係数）:**

| 地域 | PT_TO_KM | 導出根拠 |
|------|-----------|---------|
| 青森 | 0.17 | 1次変電所間距離照合 |
| 岩手 | 0.22 | 同上 |
| 秋田 | 0.19 | 同上 |
| 宮城 | 0.10 | 同上 |
| 山形 | 0.14 | 同上 |
| 福島 | 0.14 | 同上 |
| 新潟 | 0.23 | 同上 |

### Y-bus アドミタンス行列

各送電線を Pi モデルで表現し、ノード方程式の係数行列として構築:

```
Y[i,i] += y_series + jb/2   (対角: 自己アドミタンス)
Y[i,j] -= y_series           (非対角: 相互アドミタンス)
```

基準: 100MVA, 66kV (Z_base = 43.56 Ω)

---

## 8. 既知の課題・制限事項

| 課題 | 内容 | 影響 |
|------|------|------|
| 線路長精度 | 系統図は模式図のため ±30% の誤差 | インピーダンス値の精度 |
| 変電所名の欠落 | OCRが困難な箇所は空欄 | ノード識別 |
| 33kV線の過少検出 | 閉じたパス（変圧器バンク等）が除外される | 配電系統の表現 |
| 県境接続の未検出 | lat/lon が不正確な地域で欠落 | 統合モデルの連結性 |
| 電力設定の仮置き | 実際の需要・発電量は不明 | 潮流計算の絶対値 |
